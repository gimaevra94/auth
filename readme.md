# Сервис авторизации

Проект представляет из себя сервис авторизации, написанный на **Go**.  
Поддерживает регистрацию, авторизацию и вход через **Yandex OAuth**.

## Особенности

- **Регистрация** через подтверждение email кодом  
- **Авторизация** по логину и паролю  
- В каждом случае выдается токен обновления и временный ID (токен доступа)  
- Временный ID и токен обновления отзываются на всех устройствах при подозрительной активности
- Уведомления о входе с новых устройств
- Уведомления о подозрительной активности  
- Возможность смены пароля  

## Архитектура

- **База данных**: MySQL (без внешних и первичных ключей)  
- **Сессии**: Gorilla с односторонним шифрованием  
- **Контейнеризация**: Docker Compose  
- **Безопасность**: самоподписные SSL-сертификаты
